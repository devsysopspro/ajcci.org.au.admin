# OnePlaceCMS

## Install

```
$ yarn install
$ yarn production  OR  $ yarn dev
$ composer install
$ cp .env.example .env
$ php artisan key:generate
$ php artisan storage:link
$ php artisan migrate
$ php artisan db:seed

$ php artisan serve
```
